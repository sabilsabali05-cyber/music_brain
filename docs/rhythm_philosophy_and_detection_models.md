# Rhythm Philosophy and Detection Models

## Rhythm as Cycle

Indian tala frameworks emphasize cyclic return, weighted arrival points, and contrastive empty/light zones. Concepts such as sam and khali model orientation across long-form rhythmic time rather than only local beat events.

Detection targets:
- cycle length candidates and return periodicity
- return point confidence
- empty/light region identification
- phrase-resolution behavior

## Rhythm as Timeline/Key

Afro-Caribbean timeline practices (for example clave logic) frame rhythm as a guiding asymmetrical key pattern. The timeline can remain implicit while shaping where anchors, tension, and release occur.

Detection targets:
- repeated asymmetrical skeletons
- 3-2 and 2-3 fit proxies
- timeline alignment confidence
- offbeat anchor persistence

## Rhythm as Cross-Rhythm

African-diasporic layered time often supports simultaneous pulse grids (2/3/4 overlays). Cross-rhythm is not noise; it is a structured relation between coexisting metric projections.

Detection targets:
- multiple pulse-grid plausibility
- 2-against-3 and 3-against-4 tension proxies
- coexistence of incompatible subdivision families
- persistence of layered periodicity

## Rhythm as Body/Attention

Entrainment and expectation-based listening models treat rhythm as predictive coupling between perceived pulse and incoming events. Surprise and stabilization are both meaningful signals.

Detection targets:
- pulse confidence / stability
- expectation-violation estimates
- syncopation / offbeat displacement
- entrainment stability trends

## Rhythm as Everyday Life

Rhythmanalysis distinguishes cyclical and linear processes and asks whether layers produce eurhythmia (coherence) or arrhythmia (friction). Applied to performance, this can capture ritual build-up and collective arrival.

Detection targets:
- ritual build and collective arrival cues
- cyclical vs linear layer balance
- stable/unstable social groove proxies
- speech-to-song transition tendencies

## Rhythm as Hierarchy

Rhythm operates across onset, beat, bar, phrase, and section levels. Useful analysis should preserve this hierarchy rather than collapsing all timing into one scalar.

Detection targets:
- onset-level activity
- beat-level regularity
- phrase-level repetition
- section-level temporal profile shifts

## Rhythm as Geometry

Circular and necklace-based rhythm views model rotational equivalence, evenness, and spacing structure. This helps detect motifs that are equivalent up to phase.

Detection targets:
- onset circularity
- rotation-equivalent motif matches
- Euclidean evenness proxies
- circular-pattern similarity

## Rhythm as Gesture

Rhythm can be heard as embodied movement: attack/release, push/pull, lift/weight, and swell/decay. Velocity and timing shape become first-class evidence.

Detection targets:
- velocity-accent contours
- attack curve shape
- push/pull timing drift
- density swell and release trajectories

## Practical Notes

- This ontology is intentionally heuristic and should avoid overclaiming.
- "Confidence" should reflect evidence quality, not just pattern frequency.
- Sparse passages may still carry high conceptual value (for example call-response space or khali-like lightness).
