# Source Separation Witness Local Setup

This vertical slice adds a **witness-only** Demucs scaffold.

## Safety Defaults

- Demucs is disabled by default.
- No audio processing runs in setup checks, smoke checks, or planning.
- No stem files are generated by default.
- No model weights are downloaded.
- No model training occurs.
- Stems are treated as weak evidence, not ground truth.

## Local Opt-In Setup (No Execution)

1. Copy `config/model_integrations/model_integrations.example.json` to `config/model_integrations/model_integrations.local.json`.
2. In the local config only, update `models.demucs` fields:
   - `enabled`
   - `package_required`
   - `model_name`
   - `device`
   - `smoke_test_enabled`
   - `output_dir`
   - `stem_policy`
3. Keep `stem_policy` as `weak_evidence_not_truth`.
4. Keep local config files out of commits.

## Witness Scope

- Intended use: arrangement and texture inspection helper.
- Not a truth source and not training-ready by default.
- Any future separation command must be explicit and opt-in.
