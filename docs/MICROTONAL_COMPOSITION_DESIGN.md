# Microtonal Composition Design

Local-only planning layer for microtonal composition.

- No audio generation
- No cloud calls
- No training
- No model downloads
- No Ableton requirement

Supported presets:
`12_tet`, `19_edo`, `22_edo`, `24_edo`, `31_edo`, `53_edo`, `just_major_5_limit`, `just_minor_5_limit`, `blues_leaning_microtonal`, `pelog_placeholder`, `slendro_placeholder`, `maqam_placeholder`, `raga_shruti_placeholder`.

Supported export strategies:
standard MIDI + pitch bend, multichannel pitch-bend MIDI, MPE MIDI, Scala `.scl`, `.kbm` mapping, tuning JSON table, Ableton/Max tuning bridge plan, rendered-audio fallback plan.
