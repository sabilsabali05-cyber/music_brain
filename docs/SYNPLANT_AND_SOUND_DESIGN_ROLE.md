# Synplant and Sound Design Role

- Synplant is an optional downstream sound-design tool, not part of core understanding/ranker truth.
- Synplant behavior must never be faked in reports or status.
- Source-understanding and taste-learning loops can run fully without Synplant.
- Any Synplant activity must remain local/manual unless explicitly configured and authorized.
- No private paths, plugin paths, or raw media are allowed in committed Synplant-facing outputs.
